# ALX Listing App

This project is an Airbnb clone listing page built with Next.js, TypeScript, and Tailwind CSS.  

## Project Structure

- `components/` - Reusable UI components (Card, Button, etc.)
- `interfaces/` - TypeScript interfaces for props and data types
- `constants/` - Project constants and sample data
- `public/assets/` - Images, SVGs, and other assets

## Getting Started

1. Install dependencies:

"npm install"

2. Run the development server:

"npm run dev"

3. Open http://localhost:3000
 to view the project in the browser.
